# ExamenUnidad3

Examen Aplicativo de Evaluación en C++

En este repositorio se muestra el proyecto en un programa en C++ que permite realizar operaciones matemáticas básicas y registrar notas de estudiantes.  
El objetivo es practicar el uso de entrada y salida de datos, estructuras de control y manejo de archivos.

Objetivos
1. Realizar operaciones: suma, resta, multiplicación y división.
2. Registrar 5 notas por estudiante.
3. Calcular promedio, nota mayor, nota menor, aprobados y reprobados.
4. Guardar los resultados en un archivo llamado Resultados.txt.

Funcionamiento
El programa muestra un menú con tres opciones principales:
1. Operaciones básicas entre dos números.
2. Registro de notas de un estudiante.
3. Guardar resultados en el archivo Resultados.txt.

Cada vez que se guarda, la información se añade al archivo en modo acumulativo.  
De esta forma se pueden registrar los reportes de los 30 estudiantes del paralelos.
